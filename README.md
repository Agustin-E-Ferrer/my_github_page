# 🚀 Agustín Ferrer — Portfolio Personal

> **Full Stack Developer Junior** | Córdoba, Argentina 🇦🇷

Portfolio profesional desarrollado con HTML5, CSS3 y JavaScript Vanilla. Desplegado en GitHub Pages.

🌐 **[Ver sitio en vivo](https://agustin-e-ferrer.github.io/my_github_page/)**

---

## ✨ Características

- **Hero animado** — Canvas de partículas conectadas con física, efecto typed que alterna entre roles
- **Estética Cyberpunk Neon** — Paleta oscura con azul eléctrico, violeta y cyan
- **Órbita de tecnologías** — Animación CSS pura con íconos girando en dos órbitas
- **Skill bars animadas** — Se activan al hacer scroll con IntersectionObserver
- **Timeline de formación** — Línea temporal interactiva con animaciones AOS
- **Proyectos reales** — Cards con info extraída de los repositorios
- **Contadores animados** — Stats con ease-out-cubic al entrar en viewport
- **Formulario de contacto** — Genera mailto automáticamente
- **100% Responsive** — Mobile first, funciona en cualquier dispositivo
- **Sin backend** — Compatible con GitHub Pages out of the box

---

## 🛠️ Stack tecnológico

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura semántica y accesible |
| CSS3 | Diseño, animaciones, variables CSS, Grid, Flexbox |
| JavaScript Vanilla | Partículas, typed effect, scroll animations, filtros |
| AOS Library | Animaciones al hacer scroll |
| Font Awesome 6 | Iconografía |
| Google Fonts | Space Grotesk + Inter + JetBrains Mono |
